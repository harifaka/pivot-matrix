# Pivot Roadmap

## Near term

- richer task editing UX and keyboard actions
- drag-and-drop polish and section operations
- import/export workflows
- recurring task support

## Mid term

- advanced filters and saved views
- command palette expansion
- attachment/backlink support
- diagnostics and in-app recovery tools

## Long term

- plugin-style extension points
- optional sync adapters (kept offline-first by default)
- deeper analytics and planning insights
